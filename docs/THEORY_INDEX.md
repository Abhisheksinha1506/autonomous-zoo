# Theory Index

A comprehensive guide to the mathematical concepts powering the Autonomous Zoo.

## Information Theory

### Shannon Entropy
- **Project**: [Shannon Entropy Pruner](../projects/tier1-immediate/shannon-entropy-pruner/)
- **Formula**: `H(X) = -Σ p(x) * log₂(p(x))`
- **Concept**: Measures average information content in a message
- **Application**: Pruning redundant files to maximize information density

### Kolmogorov Complexity
- **Project**: [Kolmogorov Compressor](../projects/tier1-immediate/kolmogorov-compressor/)
- **Concept**: The shortest program that produces a given output
- **Application**: Self-minimizing code that maintains functionality

## Chaos Theory

### Lorenz Attractor
- **Project**: [Strange Attractor Drifter](../projects/tier1-immediate/strange-attractor-drifter/)
- **Equations**: 
  - `dx/dt = σ(y - x)`
  - `dy/dt = x(ρ - z) - y`
  - `dz/dt = xy - βz`
- **Concept**: Deterministic chaos with butterfly-wing trajectories
- **Application**: Files drift along chaotic but bounded paths

## Pattern Formation

### Turing Patterns (Reaction-Diffusion)
- **Project**: [Reaction-Diffusion Spots](../projects/tier1-immediate/reaction-diffusion-spots/)
- **Model**: Gray-Scott equations
- **Concept**: How chemical reactions create animal coat patterns
- **Application**: Leopard spots and zebra stripes in file distributions

## Statistical Mechanics

### Phase Transitions
- **Project**: [Thermodynamic Phase Changes](../projects/tier1-immediate/thermodynamic-phase-changes/)
- **Model**: Ising model
- **Concept**: Sudden state changes at critical points (solid/liquid/gas)
- **Application**: Repository undergoes phase transitions based on "temperature"

### Entropy and Equilibrium
- **Project**: [Thermodynamic Equilibrium](../projects/tier5-physics/thermodynamic-equilibrium/)
- **Law**: Second Law of Thermodynamics
- **Concept**: Systems evolve toward maximum entropy (heat death)
- **Application**: Files reach uniform "temperature" distribution

## Number Theory

### Collatz Conjecture
- **Project**: [Collatz Conjecture Explorer](../projects/tier2-mathematics/collatz-conjecture-explorer/)
- **Rule**: If even, divide by 2; if odd, multiply by 3 and add 1
- **Conjecture**: All positive integers eventually reach 1
- **Application**: Exploring trajectories of unsolved mathematical problem

### Prime Numbers
- **Concept**: Numbers divisible only by 1 and themselves
- **Applications**: Cryptography, Gödel numbering

### Partition Function
- **Project**: [Partition Function Counter](../projects/tier7-pure-math/partition-function-counter/)
- **Concept**: Number of ways to write n as a sum of positive integers
- **Formula**: Hardy-Ramanujan asymptotic formula
- **Application**: Combinatorial explosion visualization
